## Главный принцип — «коллекции»
Один поставщик = одна коллекция со своим именем. Фото разных поставщиков
сняты в разных стилях → в одной сетке НИКОГДА не смешиваем. Внутри коллекции
стиль фото одинаковый — «каша» исключена по построению.

## Страница «Облицовочный кирпич»
1. Сверху «Подберите по цвету» — цветные кружки (красный, коричневый, бежевый,
   серый, графит, тёмный, персик…). Выбор цвета → товары группами ПО коллекциям
   (стили фото не смешиваются в одну сетку).
2. Ниже — 5 коллекций лентами: имя, фраза «для кого», «от X ₽», 4–6 карточек,
   кнопка «Вся коллекция (N цветов)».
3. Страница коллекции: полная сетка + фильтры: цвет, фактура (гладкий/руст/
   береста/антик/кроста/старый город), формат (0.7НФ/1НФ/1.4НФ/евро/WDF/Ригель).
4. Лестница цен «эконом → премиум» видна сразу — клиент сам находит бюджет.
