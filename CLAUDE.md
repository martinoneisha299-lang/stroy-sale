# Проект «Строй-Сейл» — сайт стройматериалов

Интернет-каталог стройматериалов (тротуарная плитка, облицовочный кирпич, газоблок, профнастил, штакетник, сыпучие). Цель — заявки/звонки с таргета VK Реклама и Яндекс Директ. Аудитория: частные застройщики, прорабы, бригады (Россия).

## Пользователь
Новичок без опыта в коде и дизайне. Claude выполняет роли дизайнера и кодера полностью. Объяснять простым языком, по шагам, без жаргона. Решения предлагать готовыми с рекомендацией.

## ЗАКОНЫ ДИЗАЙНА (обязательны в каждой сессии)
1. Перед любой работой над дизайном/версткой ОБЯЗАТЕЛЬНО использовать скиллы:
   - `frontend-design` — официальный скилл Anthropic; задаёт «точку зрения», банит generic-шрифты и AI-штампы, требует «signature element»;
   - `ui-ux-pro-max` — подбор стиля, палитры, шрифтов, UX-правил;
   - `hallmark` — контроль качества «не выглядит сгенерированным» (audit после вёрстки);
   - `banner-design` — для акционных баннеров и рекламных креативов.
2. НИКАКИХ эмодзи в интерфейсе — только SVG-иконки (стиль Lucide, stroke).
3. Шрифты только с поддержкой кириллицы.
4. Mobile-first: трафик с таргета в основном мобильный. Каждую правку проверять на 375px и 1280px.
5. После правок вёрстки — проверка скриншотом через preview-сервер (launch.json → `prototype`, порт 8765).

## РАБОЧИЙ ПРОЦЕСС «сайт за $10k» (по гайду Metics Media, адаптирован)
Порядок для любого крупного редизайна:
1. **Референсы прежде всего.** До вёрстки взять 3–5 скриншотов/ссылок сайтов, которые нравятся пользователю (Dribbble, Awwwards, Pinterest, реальные производители стройматериалов). Прогнать через `hallmark study <URL>` → извлечь ДНК (структура, шрифты, цвета). Без референсов результат всегда «дешёвый» — это главный недостающий вход.
2. **Бриф + уточняющие вопросы.** Перед сборкой задать пользователю уточняющие вопросы и предложить 2–3 конкретных стилевых направления на выбор (как в видео). Не начинать вёрстку, пока направление не выбрано.
3. **Грейдинг по 8 столпам $10k-сайта** (проверять ПОСЛЕ каждого крупного этапа, честно):
   `point of view · typography · colour · hierarchy · imagery · motion · mobile · «невидимое» (скорость/отполированность)`.
   Группы: вкус / содержание / ощущение качества.
4. **Полировка: «дороже, а не пестрее».** Правки просить пачками через ИНТЕНТ, не через перечень («нужны рукотворные микровзаимодействия, нижние секции пресноваты — не делать пестрее, сделать дороже»), а не «добавь тень сюда».
5. **Картинки/видео.** Лучшее — реальные фото товара от заказчика. Нет фото → AI-генерация (ChatGPT image / Вео через агрегатор). Промпт для картинки просить у самого Claude — он знает проект.
6. **Готовые компоненты — 21st.dev** (бесплатно): анимации, скролл-эффекты, hero. Копировать «prompt» с компонента и отдавать Claude. ВАЖНО: прототип — статический HTML, React-компоненты не вставлять как есть (Claude сам отклонит и перепишет под ванильный HTML).

## Требования из ТЗ (переписка с заказчиком)
- Корзина без регистрации; заказ = звонок менеджера за 5 минут; оплата при получении.
- Калькуляторы: кирпич (стены), кровля, газоблок; плитка по площади в карточке товара.
- Акционные баннеры: задаётся срок действия, после даты баннер исчезает сам.
- Характеристики товара — простым языком «где используется», без перегруза таблицами.
- Единый телефон в шапке + «перезвоним за 5 минут».
- Реклама ведётся на страницы категорий, не на главную.
